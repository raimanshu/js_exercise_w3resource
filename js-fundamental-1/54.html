Write a JavaScript program to count the number of vowels in a given string.

Use Array.from() to create an array of the desired length. Use (end - start + 1)/step and a map function to fill the array with the desired values in the given range. Omit the second argument, start, to use a default value of 0. Omit the last argument, step, to use a default value of 1.
