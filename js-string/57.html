Points to remember

Write a JavaScript function that receives a string and determine whether or not it can be rearranged to become a palindrome. Go to the editor When a word, phrase, or sequence can be read both forward and backward, it is considered a palindrome. e.g., madam or nurses run. Test Data : ("maamd") -> true ("civic") -> true ("IO") -> false (12321) -> "It must be a string."
