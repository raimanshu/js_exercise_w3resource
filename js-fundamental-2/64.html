Write a JavaScript program to Initializes an array containing the numbers in the specified range (in reverse) where start and end are inclusive with their common difference step Use Array.from(Math.ceil((end+1-start)/step)) to create an array of the desired length(the amounts of elements is equal to (end-start)/step or (end+1-start)/step for inclusive end), Array.prototype.map() to fill with the desired values in a range. Omit the second argument, start, to use a default value of 0. Omit the last argument, step, to use a default value of 1.
